License Plate Recognition

Completed 2024
🚗

Overview

Real-time automatic license plate recognition (ALPR) system. YOLO detects and localizes license plates in each video frame, and EasyOCR extracts the alphanumeric characters. The pipeline runs in real time on video streams.

YOLO EasyOCR Python OpenCV Real-time
GitHub →
Category Computer Vision · OCR
Input Video stream / image

Detailed Description

The ALPR pipeline processes each video frame through a two-stage approach: first, YOLO localizes license plate regions with bounding boxes; then EasyOCR reads the characters within each detected region.

Post-processing filters false positives and cleans OCR output using regex patterns matching local license plate formats. Results are overlaid on the video in real time.
