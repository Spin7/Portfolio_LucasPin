Overview

Development and deployment of a client-server web application with a full frontend (HTML/CSS/JS) and backend (Python/FastAPI), deployed on Vercel (frontend) and Railway (backend). The project demonstrates end-to-end web development skills.

Detailed Description

The application follows a client-server architecture. The frontend is a static site served by Vercel, communicating with a FastAPI backend hosted on Railway via HTTP requests.

This project demonstrates the full development lifecycle: UI design, API design, database integration, deployment pipelines, and CI/CD configuration.
