Autonomous Hexapod Robot

Completed 2024 – 2025
🤖

Overview

Development of an autonomous hexapod robot integrating computer vision for obstacle and target detection, autonomous navigation via ROS2, and physical simulation in Gazebo for testing and validation before real-world deployment.

ROS2 Gazebo C++ Python OpenCV YOLO

Institution Universidad Nacional de Asunción
Context Robotics Club · Faculty of Engineering

Detailed Description

This project involved the full development pipeline of an autonomous hexapod robot — from mechanical design to software integration. The hexapod uses a gait controller that generates smooth, stable walking patterns across different terrains.

The computer vision subsystem uses YOLO to detect obstacles and targets in real time. ROS2 handles the communication between perception, planning, and actuation nodes, enabling fully autonomous behavior without human intervention.

Gazebo was used extensively for simulation — testing gaits, navigation algorithms, and sensor integration in a physics-accurate environment before deploying to hardware.
