IoT Intelligent Mosquito Trap

Active 2025 – 2026

Overview

Design and implementation of an IoT architecture based on HTTP and PostgreSQL for the registration, storage, and analysis of biological detections in intelligent mosquito traps. The system supports remote data collection and real-time monitoring.

Detailed Description

The intelligent mosquito trap integrates edge computing hardware that captures biological data from the trap environment. This data is transmitted via HTTP to a central server and stored in a PostgreSQL database for analysis.

The backend provides REST API endpoints for data ingestion, querying, and visualization. The system is designed to scale across multiple traps deployed in different locations simultaneously.
