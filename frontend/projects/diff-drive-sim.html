Overview

Simulator for differential-drive robots built with PyGame (graphics) and PyMunk (2D physics), incorporating trajectory planning via RRT (Rapidly-exploring Random Trees) for autonomous navigation in obstacle-filled environments.

Detailed Description

The simulator provides a 2D physics environment where a differential-drive robot navigates around obstacles. PyMunk handles collision detection and rigid body dynamics. PyGame renders the environment in real time.

RRT builds a tree from the start position, randomly sampling the configuration space and extending toward sampled points. When the goal is reached, the path is extracted and followed by the robot using a PID controller for heading and speed.
