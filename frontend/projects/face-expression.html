Face Expression & Tracking

Completed 2024
😶

Overview

System for real-time facial expression recognition and person tracking using YOLO (ONNX Runtime) for detection and MediaPipe Face Landmarks for expression classification. The system can identify emotions and track multiple persons simultaneously in a live video stream.

YOLO ONNX Runtime MediaPipe Python OpenCV
GitHub →
Category Computer Vision · AI
Runtime Real-time (ONNX)

Detailed Description

The system uses YOLO exported to ONNX format for efficient, cross-platform person detection. ONNX Runtime provides fast inference without framework dependencies, making the system portable and performant on both CPU and GPU.

MediaPipe Face Landmarks extracts 468 facial keypoints per face, enabling a lightweight classifier to distinguish between emotional states such as neutral, happy, surprised, and others. Person tracking maintains consistent IDs across frames.
