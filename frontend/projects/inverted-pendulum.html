Self-Balancing Inverted Pendulum

Completed2023
⚖️

Overview

Design, modeling, and implementation of a state-feedback controller for a self-balancing inverted pendulum robot. The system uses LQR (Linear Quadratic Regulator) control derived from a linearized state-space model, validated in MATLAB/Simulink before embedded implementation.

Detailed Description

The inverted pendulum is linearized around its upright equilibrium. A state-space model is derived from Newton-Euler equations, and LQR is used to compute optimal feedback gains that minimize a weighted cost function of state deviations and control effort.

An IMU (MPU6050) provides angle and angular velocity feedback. The computed control signal drives DC motors via an H-bridge. The controller runs on a microcontroller at fixed sample rate.
